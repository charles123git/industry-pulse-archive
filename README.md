# Industry Pulse Archive

Daily industry pulse scans from Reddit — trends, sentiment shifts, and unmet needs.

Generated by the [AEO Platform](https://github.com/BlueBoobyAI/aeo) Reddit Industry Pulse skill, using the Parameter Dimensions framework (Source × Transform × Gate × Output × Loop).

## How to Browse

- `pulse/{industry}/YYYY-MM-DD.md` — human-readable briefs
- `pulse/{industry}/YYYY-MM-DD.json` — structured data (JSON)
- `index.json` — master index of all pulses

## Process

Each pulse fetches hot/new/rising posts from industry subreddits, applies an Analyst pass (velocity/volume/patterns) then an Empath pass (sentiment/unmet needs/problems), scores findings via a >60 threshold with Skeptic verification, and stores everything for Predictive Coding.
